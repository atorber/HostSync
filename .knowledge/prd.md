# 配置管理工具

## 概述

- 支持对AI APIkey管理
- skill管理
- claude code、openclaw、codex等配置文件进行管理
- 支持本地管理和云端同步，本地管理不需要登录，云端管理使用S3的配置信息登录

## 产品设计

### UI界面

1. 左侧菜单栏，包含菜单：
   1.  API管理：增删改查API key以及接入点等信息
   2.  Skill管理：管理所有的Skill，支持扫描本地所有的目录，发现skill、skills命名的文件夹，管理文件夹下的skill.md文件
   3.  配置管理：claude code、openclaw、codex等配置文件进行管理，支持自动扫描本地目录发现配置文件
2.  内容区域：以列表+右侧抽屉显示详情

### 功能描述

1. API管理
   1. 支持增删改查配置
   2. 支持将配置同步到S3存储
   3. ...
2. Skill管理
   1. 支持Skill的增删改查
   2. 支持自动扫描本地目录，发现skill、skills命名的文件夹，管理文件夹下的skill.md文件
3. 配置管理
   1. 管理本地claude code、openclaw、codex等配置文件
   2. 支持自动扫描本地目录发现配置文件
4. 云端同步
   1. 支持设置、修改S3存储配置
   2. 支持一键备份本地配置到云端
